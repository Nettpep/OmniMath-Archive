# 📖 คู่มือเขียนโจทย์แบบเห็นภาพ

> [!abstract] หัวใจสำคัญ
> โจทย์ที่ดีไม่ใช่แค่ "ทำได้" แต่ต้อง **เข้าใจว่าทำไม** — ใช้โครงสร้างนี้เพื่อให้ลูกหรือผู้เรียนเห็นภาพและคิดเป็น

## 🎯 โครงสร้างวิธีคิด (Solution)

### 1️⃣ ตั้งคำถาม "ทำไม?" ก่อน
```
> [!question] ทำไมต้องทำแบบนี้?
> อธิบายเหตุผล — ทำไมวิธีอื่นไม่ได้? ทำไมต้องขั้นตอนนี้?
```

**ตัวอย่าง:** ทำไมต้องทำส่วนให้เท่ากัน? เพราะชิ้นเค้ก 1/3 กับ 1/4 ขนาดไม่เท่ากัน — เทียบตรงๆ ไม่ได้!

### 2️⃣ อธิบายหลักการ/นิยาม
```
> [!info] [หัวข้อ] คืออะไร?
> นิยามสั้นๆ ที่เข้าใจได้
```

### 3️⃣ แสดงวิธีทำทีละขั้น
- แต่ละขั้นเขียนว่า **ทำอะไร** และ **ทำไม**
- ใช้สูตร/สมการประกอบ

### 4️⃣ เพิ่ม "เห็นภาพ" ด้วยตัวอย่างจริง
```
> [!example] 🔥 เห็นภาพ
> เปรียบเทียบกับของใกล้ตัว — พิซซ่า, เงิน, แอปเปิล, ลูกอม
```

### 5️⃣ วิธีลัด (ถ้ามี)
```
> [!tip] 🌱 วิธีลัด
> สูตรหรือเทคนิคที่เร็วขึ้น — แต่ต้องเข้าใจหลักก่อน
```

## 📋 Callouts ที่ใช้

| Callout | ใช้เมื่อ |
|---------|----------|
| `> [!question]` | ตั้งคำถาม "ทำไม?" |
| `> [!info]` | อธิบายหลักการ/นิยาม |
| `> [!example]` | ตัวอย่างเห็นภาพ |
| `> [!tip]` | วิธีลัดหรือเคล็ดลับ |
