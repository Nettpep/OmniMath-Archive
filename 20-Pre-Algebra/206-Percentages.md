# 🪐 ร้อยละ (Percentages)

> [!abstract] หัวใจสำคัญ
> ร้อยละ = ส่วนใน 100 — ใช้คำนวณส่วนลด ภาษี ดอกเบี้ย และเปรียบเทียบข้อมูล การแปลงระหว่าง %, เศษส่วน และทศนิยมต้องคล่อง

## 📜 กฎและสูตร (The Laws)

### นิยาม
$$x\% = \frac{x}{100} = 0.0x$$

### การคำนวณ
$$x\% \text{ ของ } A = A \times \frac{x}{100}$$

### โจทย์ประเภท "A เป็นกี่% ของ B"
$$\frac{A}{B} \times 100 = ?\%$$

### การเพิ่ม/ลดร้อยละ
$$A \text{ เพิ่ม } 20\% = A \times 1.20$$
$$A \text{ ลด } 15\% = A \times 0.85$$

## 💡 วิธีการมอง (Mental Model)

> [!example] สำหรับเด็ก: ส่วนลด
> ลด 25% = จ่าย 75% = คูณ 0.75

> [!example] การเปรียบเทียบ
> เกรด 8/10 = 80% — แปลงเป็น % เพื่อเปรียบเทียบง่าย
