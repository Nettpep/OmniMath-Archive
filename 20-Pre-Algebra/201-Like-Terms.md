# 🪐 พจน์ที่คล้ายกัน (Like Terms)

> [!abstract] หัวใจสำคัญ
> พจน์ที่คล้ายกันคือพจน์ที่มีตัวแปรและเลขชี้กำลังเหมือนกัน — เราสามารถรวมพวกมันได้โดยการบวกลบสัมประสิทธิ์ เพราะพวกมันเป็น "ของชนิดเดียวกัน" เหมือนแอปเปิลกับแอปเปิล

## 📜 กฎและสูตร (The Laws)

### คำจำกัดความ
**Like Terms** = พจน์ที่มีตัวแปรเดียวกัน และเลขชี้กำลังของตัวแปรเท่ากัน

$$3x + 5x = 8x \quad \text{แต่} \quad 3x + 5y \neq 8xy$$

### กฎการรวมพจน์
$$ax^n + bx^n = (a+b)x^n$$

เมื่อ $a, b$ เป็นสัมประสิทธิ์ และ $x^n$ เป็นส่วนตัวแปรที่เหมือนกัน

### ตัวอย่าง
$$2x^2 - 5x + 3x^2 + 7x = (2+3)x^2 + (-5+7)x = 5x^2 + 2x$$

## 💡 วิธีการมอง (Mental Model)

> [!example] เปรียบเทียบ: แอปเปิลกับส้ม
> - $3x + 5x$ = แอปเปิล 3 ลูก + แอปเปิล 5 ลูก = แอปเปิล 8 ลูก ✓
> - $3x + 5y$ = แอปเปิล 3 ลูก + ส้ม 5 ลูก = ต้องเก็บแยกกัน ไม่สามารถรวมเป็น "8 ผล" ได้ ✗

> [!example] เทคนิคการจัดกลุ่ม
> วงกลมหรือขีดเส้นใต้พจน์ที่คล้ายกันก่อน แล้วค่อยรวม — ลดโอกาสพลาดเครื่องหมาย (โดยเฉพาะลบ)
