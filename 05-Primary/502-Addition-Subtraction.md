# 🪐 การบวกและการลบ (Addition & Subtraction)

> [!abstract] หัวใจสำคัญ
> บวก = รวมกัน ลบ = แยกออก — การเข้าใจความหมายก่อนสูตรจะทำให้ลูกคิดเป็นและไม่ท่องจำ

## 📜 กฎและสูตร (The Laws)

### การบวก
$$a + b = c \quad \text{(รวม a กับ b ได้ c)}$$

### การลบ
$$c - a = b \quad \text{(เอา a ออกจาก c เหลือ b)}$$

### ความสัมพันธ์
$$a + b = c \quad \Leftrightarrow \quad c - a = b \quad \Leftrightarrow \quad c - b = a$$

## 💡 วิธีการมอง (Mental Model)

> [!example] สำหรับเด็ก: เรื่องเล่า
> บวก = "มี 3 แอปเปิล ได้เพิ่ม 2 ลูก ตอนนี้มีกี่ลูก?" (3+2=5)
> ลบ = "มี 5 แอปเปิล ให้ไป 2 ลูก เหลือกี่ลูก?" (5-2=3)

> [!example] ใช้รูปภาพและของจริง
> วาดวงกลม, ใช้ลูกปัด, นับนิ้ว — ให้เห็นภาพก่อนคำนวณ
