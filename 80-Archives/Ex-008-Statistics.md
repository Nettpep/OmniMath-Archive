# 📝 บันทึกปริศนา: สถิติ

## ❓ โจทย์ (The Question)

> [!quote]
> ข้อมูลคะแนน 5 คน: 70, 80, 85, 90, 75 จงหาค่าเฉลี่ยและความแปรปรวน

## 🕵️ วิธีการพิชิต (Solution)

$$\bar{x} = \frac{70+80+85+90+75}{5} = \frac{400}{5} = 80$$

$$\sigma^2 = \frac{(70-80)^2 + (80-80)^2 + (85-80)^2 + (90-80)^2 + (75-80)^2}{5}$$
$$= \frac{100 + 0 + 25 + 100 + 25}{5} = \frac{250}{5} = 50$$
