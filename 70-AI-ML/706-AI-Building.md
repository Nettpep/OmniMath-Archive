# 🪐 การสร้างปัญญาประดิษฐ์ (AI Building)

> [!abstract] หัวใจสำคัญ
> การสร้าง AI คือการรวมความรู้ทั้งหมด — ตั้งแต่การเตรียมข้อมูล เลือกโมเดล train ประเมิน และ deploy — เป็นจุดหมายปลายทางของเส้นทางเรียนรู้คณิตศาสตร์สู่ Data Science และ ML

## 📜 กฎและสูตร (The Laws)

### ML Pipeline
1. **Problem Definition** — กำหนดปัญหาและ metric
2. **Data Collection** — รวบรวมข้อมูล
3. **Data Preparation** — [[60-Data-Science/601-Data-Analysis|601-Data-Analysis]]
4. **Model Selection** — Regression, Classification, Clustering, Deep Learning
5. **Training** — Gradient Descent, Backpropagation
6. **Evaluation** — Accuracy, MSE, F1, etc.
7. **Deployment** — API, App

### เครื่องมือ
- **Python:** NumPy, Pandas, Scikit-learn, PyTorch/TensorFlow
- **คณิตศาสตร์:** [[40-Geometry-Trig/402-Matrices|402-Matrices]], [[50-Advanced/501-Derivatives|501-Derivatives]], [[50-Advanced/502-Probability|502-Probability]]

## ⛓️ เส้นใยความสัมพันธ์ (Dependencies)

- **ต้องรู้ก่อน:** [[70-AI-ML/701-Linear-Regression|701-Linear-Regression]], [[70-AI-ML/702-Neural-Networks|702-Neural-Networks]], [[70-AI-ML/703-Classification|703-Classification]], [[70-AI-ML/705-Deep-Learning|705-Deep-Learning]]
- **นำไปสู่:** โครงการจริง — Chatbot, Image Recognition, Recommendation System

## 💡 วิธีการมอง (Mental Model)

> [!example] เปรียบเทียบ: สร้างบ้าน
> รากฐาน (คณิตศาสตร์) → โครงสร้าง (Data Science) → ระบบไฟฟ้า (ML) → บ้านสมบูรณ์ (AI Product)

> [!example] เริ่มจากอะไร?
> โครงการเล็ก — ทำนายราคาบ้าน, แยกรูปแมว/สุนัข, ทำ sentiment analysis — แล้วค่อยขยาย
